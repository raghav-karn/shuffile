About

Welcome to Shuffile by Raghav Karn. Shuffile is a end-to-end files sharing app meant for quick sharing of small files like small images, gifs, text files and more.

You can use Shuffile to create password-protected links to share, or transfer files directly to another user on Shuffile with hands-free ease! There is no hard limit on the file size, but speed drastically decreases for files of sizes more than 10MB.
